# Emotion_and_image_based_music_recommendation_system


'''This project is based on smart music recommendation system that suggests songs based on the user’s 
emotions .Using Deep learning algorithms and models , the system analyzes facial expressions or 
images to detect emotions such as happiness, sadness, anger, disgust, surprise, neutral and fear. Based 
on the current mood the system recommends which song should be played and preferred by the user. 
This enhances user's experience by providing a personalised emotion based music recommendation 
system.'''
